# User Management App

A simple web application for managing user data.

## Features

- Add new users with unique IDs.
- View a list of users with pagination.
- Update user details.
- Delete users.

## Technologies Used

- React.js for the frontend.
- Node.js with Express for the backend.
- MySQL for database storage.
